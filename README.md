# Entre Amigos — Cafetería

Sitio web de la cafetería **Entre Amigos**.

Es un sitio estático. La página principal es [`index.html`](index.html).

## Estructura

- `index.html` — página principal
- `support.js`, `image-slot.js` — scripts de apoyo
- `assets/` — logos e imágenes
- `uploads/` — Manual de Marca (PDF)

## Ver el sitio localmente

Abrí `index.html` en el navegador, o serví la carpeta con un servidor estático:

```bash
python -m http.server 8000
```

Y entrá a http://localhost:8000
